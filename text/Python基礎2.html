Python基礎 2¶

〇リスト¶

リストは、複数の値をまとめて扱う場合に使用するものです。
リストの値はmutable、つまり可変的で、値を入れ替えたり、リスト毎四則演算を行うことが可能です。
リストは基本的に全体を[]で囲み、各要素は,で区切ります。
Pythonのリストは文字列でも数値でも、どのような型の値でも格納することができ、
リストの中にリストを格納することもできます。
リストの各要素へのアクセスはインデックスを使い、要素の追加や入れ替えなどが可能です。
[ ]:

# リストの作成
a = [1, 2, 3, 4, 5]

# 2番目の要素を取得
# リストの順番は 0 始まり
# 後ろからアクセスするときは -1,-2とマイナスを入れる

b = a[1]
print(b)

# 末尾に要素を追加する
a.append(6)
print(a)

# 要素の入れ替え
a[2] = 9
print(a)

〇タプル¶

タプルはリストに似ていますが、リストと違ってimmutable、つまり非可変的です。
つまり要素の追加や削除、入れ替えなどはできません。
タプルは全体を()で囲み、各要素は,で区切ります。
要素を変更したくない恒久的なデータにしたいときなどは、リストよりもタプルを使用する方がベターです。
[ ]:

# タプルの作成
a = (1, 2, 3, 4, 5)

# 2番目の要素を取得
b = a[1]
print(b)

〇if文¶

条件によって、処理の仕方を分けたいときなどに使用します。
例えばリンゴなら100円、ミカンなら80円支払うみたいに。
ifの条件が満たされていなければ、elifの条件が上から順番に判定されます。
これらの条件がすべて満たされていなければ、else内の処理が実行されます。
つまり、上から順番に一つずつ判定を処理していきます。
なので判定の記述の順序にも注意が必要です。
Pythonではブロック、つまり処理の範囲を行頭のインデントで表します。
インデントが始まって、それ以降にインデントしていない行が出現したら、
その直前にブロックは終了していることになります。
Pythonの場合、インデントには、半角スペース4つを用います。
[1]:

# 実行例
x = input('数字を入力してください: ')
x = int(x)

if x < 0 :                # もし x<0 が true なら実行
    print("xは負の数です")
elif x == 1 :             # 上記にあてはまらず、もし x==1 が true なら実行
    print("xは1です")
elif x != 2 :             # 上記に2つにあてはまらず、もし x!=2 が true なら実行
    print("xは2以外です(ただし、正の数で1ではありません)")
else:                    # 上記のどれにも当てはまらないなら実行
    print("xは2です")#消去法で2しか条件に当てはまりません

数字を入力してください: 8
xは2以外です(ただし、正の数で1ではありません)
elifは何個でも記述することが可能です。
逆にelifやelse は省略可能です。
if文 はネスト(入れ子構造、if文の中にif文の構造の事)できます。
[ ]:

if x > 0 :
    if x <= 10 :
        print("xは0～10の間だよ")
