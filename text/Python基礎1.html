Pythonの基礎１¶

Pythonについての基礎的な部分の解説となります。 解説部分を読み、直後のセルをShift + Enterで実行して実行結果を確認しながら進めてください。 自分で代入する数値を変えて動きを確かめてみても大丈夫です。

課題がありますので、課題部分にも挑戦してみてください。 練習問題もありますが、それは物足りない人向けです。

〇変数や型について¶

プログラミングにおいて数値や文字列、リストや辞書など、変数に入れる値のデータの型は複数あります。
数値と数値は計算できるけど、数値と文字列は計算できなくてエラーが出力されるなど、
自分が行おうとしている処理がデータ型によってはできないことがあります。
なので変数に入っている値のデータ型がなんなのかを抑えておきましょう。
Pythonでは以下のように、値を代入するところから記述を始めることができます。
Pythonは、変数に対して型の明示は不要です。
例えば、整数型の変数に文字列を代入すれば、それは文字列型の変数になります。
Pythonの主な型を以下に示します。

Tips1¶

演算子自体は四則演算するだけのものではありません。
基本的な演算子を下記にまとめておきます。
なお、次回以降の項目で改めて紹介するので、ここで全部完璧にする必要はありません。
1.四則演算子：主に計算する演算子のことです。計算の結果を返します。
a+b : 足し算
a-b : 引き算
a/b : 割り算
a*b : 掛け算
a%b : 割り算のあまり
2.代入演算子(他にもありますが、最初は=だけで十分です)
a = a+1 : 代入
3.比較演算子：比較し、その結果 True か False を返す演算子です。
a==b : 等しい(ならTrue)
a!=b : 等しくない(ならTrue)
a<>b : 等しくない( != と同じ)
a<b : a が bより小さい
a<=b : aがb以下
a>b : aがbより大きい
a>=b : aがb以上
※a=>b :この演算子はありません、エラーになります。
4.論理演算子：True,Falseを比較する演算子です。
a and b : aもbもTrueならTrueを返す
a or b : aまたはbがTrueならTrueを返す
not a : aがFalseならTrueを返す(反転)
