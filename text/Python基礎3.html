Python基礎 3¶

〇 for文¶

指定した回数ループするためにはfor文を用います。
ループする範囲を指定するためには、リストやrangeをin演算子とともに用います。

書き方の基本構文は以下の通りです。

for 変数 in 変数グループ :
    処理

この場合も、リストなどと同様に開始番号は0始まりとなります。

Tips¶

rangeについて：変数グループはいくつかありますが一番使うrangeを書いておきます。
range(終了) : 0から終了(終了含まず)までのグループを表します。
range(開始,終了) : 開始から終了(終了含まず)までのグループを表します。
range(開始,終了,step数) : 開始から終了(終了含まず)までのグループをstep数毎に表します。
range(5) -> [0,1,2,3,4]
range(2,5) -> [2,3,4]
range(1,10,2) -> [1,3,5,7,9]
range(5,1,-1) -> [5,4,3,2]
[ ]:

# for文の出力例
for i in range(5):  # 0,1,2,3,4 です。
    print(i)        #ループごとに i には 0,1,2,3,4 とはいります。

[ ]:

# for文の出力例2
print("1から10までで3で割り切れる数字のみ表示する")
for n in range(1,11):
    if n%3 == 0 :
        print(n)
print('-------')

〇 while文¶

条件を満たす限りループするためにはwhile文を用います。
回数が未定の待機処理のときになどに使います。
Trueになり続けてしまう条件を指定すると無限ループになるので注意が必要です。

書き方の基本構文は以下の通りです。

while 条件 :
    処理
---
「条件」が True の間、処理を繰り返します。
if文と同様にスコープがあり、ネストできます。

この場合も、リストなどと同様に開始番号は0始まりとなります。

[ ]:

# while文の出力例
i=0
print("繰り返し開始")
while i<5 :
    print(i)
    i = i+1
print("繰り返し終了")
# ポイントは while の中で i に+1 している点です。
# i は最初は 0 ですが、繰り返すうちに増えていき 5 になると i<5 が False になり while から抜けます。

練習問題 3¶

下記の最大値を出力するプログラムを参考に、target のリストの中から最大値と2番目に大きい数字を出力するプログラムを作成してください。

max = 0
target = [5,4,2,6,9,1,2,3,10,2,5]
for n in target:
    if n > max :
        max = n
print("最大値は " +max+ " です")
—実行例—
最大値は 10 です。 2番目に大きい数字は 9 です。
